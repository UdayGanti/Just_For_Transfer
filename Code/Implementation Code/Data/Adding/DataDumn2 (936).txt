About The Bull Runner
The USF bus system is known as the Bull Runner and with a valid USFCard, students,
               faculty and staff have fare free access. Visitors must be accompanied by someone with
               a valid USFCard or have a USF daily bus pass. Please refer to our web site for current
               information and route schedules.
